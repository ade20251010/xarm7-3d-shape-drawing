# README_01.md

# xArm7 工业级多工位运动控制项目

## ## 项目概述

本项目基于ROS2实现xArm7机械臂的**梯度化运动控制**，从基础抓取-放置任务逐步升级为工业级多工位循环作业，涵盖轨迹规划、平滑运动、碰撞规避、故障容错等核心工业机器人应用能力。

## ## 版本族谱（能力递进：基础→进阶→工业级）

三个版本梯度清晰、功能层层递进，从基础运动闭环逐步升级为工业级实用场景，完整覆盖机械臂运动控制核心能力，具体如下：

|**版本文件**|**定位**|**核心功能**|**技术亮点**|
|---|---|---|---|
|xarm_square_official_01.py|基础可用版（入门级）|单工位抓取+放置+归位，完成基础运动闭环|稳定运行、无额外依赖、实现ROS2基础通信与关节限位|
|xarm_square_official_02.py|进阶平滑版（提升级）|五次多项式轨迹规划、避障抬升、运动参数优化，提升运动流畅度|消除关节急跳、配置工业级运动约束、纯Python标准库实现，无numpy依赖|
|xarm_square_official_03.py|工业级完整版（终极版）|多工位循环作业（上料→检测→加工→码垛）、故障容错、紧急停止、生产数据追踪，模拟真实工厂场景|真实工厂场景适配、全流程碰撞规避、工程化容错设计、生产数据可追溯，完全贴合工业应用需求|
## ## 核心功能

### ### 1. 基础能力（v01）

- 笛卡尔空间→关节空间映射

- 轨迹发布与控制器通信

- 关节限位安全防护

- 抓取-放置-归位单任务流程

### ### 2. 进阶能力（v02）

- 五次多项式插值（Quintic Polynomial）：消除关节急跳

- 工业级运动参数：速度/加速度限制、混合时间

- 避障抬升：移动前抬升机械臂，避免碰撞

- 轨迹发布重试机制：提升通信稳定性

### ### 3. 工业级能力（v03）

- 多工位循环生产：支持自定义工位坐标、循环次数

- 碰撞规避：全流程抬升运动，适配工厂工装布局

- 故障容错：紧急停止自动归位、轨迹发布重试、关节限位告警

- 生产数据追踪：记录每轮循环时间、路点数、工位流转

- 工业级运动学：加减速曲线规划（加速→匀速→减速）

## ## 运行说明

### ### 前置条件

1. 已部署xArm7 ROS2仿真环境

2. 确保`/xarm7_traj_controller/joint_trajectory`话题可通信

3. Python 3.8+（无额外依赖，纯标准库）

### ### 运行步骤

#### #### 1. 启动仿真环境

```bash
cd ~/dev_ws
source install/setup.bash
ros2 launch avatar_challenge start.launch.py
```

#### #### 2. 运行对应版本

```bash
# 基础版
python3 ~/dev_ws/src/avatar_challenge/src/xarm_square_official_01.py

# 进阶版
python3 ~/dev_ws/src/avatar_challenge/src/xarm_square_official_02.py

# 工业级完整版（推荐演示）
python3 ~/dev_ws/src/avatar_challenge/src/xarm_square_official_03.py
```

## ## 核心设计思路

### ### 1. 坐标变换（笛卡尔→关节）

- 对X/Y/Z轴进行归一化处理（映射到[-1,1]）；

- 采用五次多项式（Quintic Polynomial）插值，保证位置/速度/加速度连续；

- 所有关节值强制限位，避免机械臂超限。

### ### 2. 轨迹规划

- 短距离：仅加减速运动（无匀速段）；

- 长距离：加速→匀速→减速完整运动曲线；

- 多工位运动：抬升→平移→下降的安全路径，避免碰撞。

### ### 3. 容错设计

- 控制器连接超时检测（8秒超时）；

- 轨迹发布3次重试；

- 紧急停止（Ctrl+C）自动归位到安全位置；

- 关节限位告警，避免机械损伤。

## ## 自定义配置（工业级版本）

### ### 1. 修改工位坐标

```python
STATIONS = {
    'FEEDER': {'x': 0.40, 'y': 0.20, 'z': 0.15, ...},  # 上料工位
    'INSPECTION': {'x': 0.50, 'y': 0.0, 'z': 0.25, ...}, # 检测工位
    # 可新增/修改工位坐标
}
```

### ### 2. 调整运动参数

```python
MOTION_PARAMS = {
    'MAX_SPEED': 0.18,        # 最大笛卡尔速度（m/s）
    'MAX_ACC': 0.12,          # 最大加速度（m/s²）
    'CYCLE_COUNT': 2          # 生产循环次数
}
```

### ### 3. 调整避障抬升高度

```python
MOTION_PARAMS['LIFT_HEIGHT'] = 0.10  # 抬升高度（m），避免碰撞
```

## ## 故障排查

### ### 1. 控制器连接超时

- 检查仿真环境是否正常启动；

- 确认`/xarm7_traj_controller/joint_trajectory`话题存在：
        `ros2 topic list | grep joint_trajectory`

### ### 2. 关节限位告警

- 降低运动速度/加速度；

- 调整工位坐标，避免关节超限；

### ### 3. 运动卡顿

- 增大`CYCLE_COUNT`外的循环间隔；

- 降低`MAX_SPEED`/`MAX_ACC`参数。
